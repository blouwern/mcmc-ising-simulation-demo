## 1.1. 实践内容介绍
本次实验使用蒙特卡洛（MCMC）方法对二维 Ising 模型进行数值模拟，比较两种常用更新算法的表现：Local update 局部翻转和 Wolff 簇翻转。目标包括：绘制热力学量（能量、磁化强度、比热 $C_v$、磁化率 $\chi$）随温度的曲线，寻找 $C_v/\chi$ 的峰值以估计临界行为，并比较两种更新方法在同一格点尺寸 (6x6) 上的量值差异与可视化表现。


## 1.2. 仿真设置与参数
- 晶格尺寸: 6 x 6
- 温度扫描范围为 0~10，间隔 0.1（脚本内使用 CSV 中的温度列）
- 每个温度点的采样由已有 C++ 程序生成，数据已保存为 CSV：
  - [ising_metro_6x6.csv](ising_metro_6x6.csv)(因书写习惯，本次实验源码中若无特殊说明，Metropolis/metro前后缀相关词均属于Local update方法)
  - [ising_wolff_6x6.csv](ising_wolff_6x6.csv)
- 后处理/绘图: Python (numpy, pandas, matplotlib)，报告中图像保存在工作目录。
- 参考临界温度（无限大格点的理论值）: $T_c = 2 / \ln(1+\sqrt{2}) \approx 2.269185$。


## 1.3. 上机结果与讨论

### 1.3.1. 使用平台说明
- 操作系统：Debian GNU/Linux 13 (trixie)
- 内核版本：Linux 6.12.74+deb13+1-amd64
- 处理器：Intel(R) Core(TM) Ultra 5 125H
- C 编译器：g++ 14.2.0
- Python 环境：Python 3.13.5 (主要包括 numpy, pandas, matplotlib)


### 1.3.2. 算法与实现要点
- Local update: 逐自旋尝试翻转，按能量改变的 Metropolis 概率接受。
- Wolff: 以簇为单位生长并整体翻转，能有效减小临界慢化（autocorrelation）。
- 数据处理: 从 CSV 读取并计算每自旋平均能量 $E/N$、绝对磁化强度 $|M|/N$、比热 $C_v/N$、磁化率 $\chi/N$；在临界附近用峰值搜索（温度窗口 1.5–3.5）找出峰位。


### 1.3.3. 程序使用说明
1. 生成/准备数据：运行 C++ 模拟，请编译并运行 `ising.c++`（源码仓库已包含生成的 CSV 文件），生成.csv数据文件。

```bash
g++ -std=c++17 ising.c++ -O2 -o ising_run
./ising_run
```

2. 使用 Jupyter Notebook 绘图与分析汇总：
在 IDE 中打开并运行本目录下的 `conclusion.ipynb`：
   - 包含的 Python 代码会读取上述两份 CSV，生成全温度范围散点图对算法差异进行直观比较。
   - 会计算并输出能量、磁化强度、比热峰值等数值，图像将分别保存为 `ising_summary_6x6_notebook.png` 与 `ising_critical_zoom_6x6.png`。

补充：结果可视化时原始两条曲线在若干温度点处重叠较严重，目前解决方法是通过减少标记密度、使用空心标记与不同 z-order改进可读性。

  2. **临界区域特写**：

     *图 2：针对 $T \in (1.4, 3.6)$ 附近临界区域的比热与磁化率响应详图，图中黑色虚线为无限系统二维临界温度 $T_c \approx 2.269$。结合散点可判断出当前有限尺寸体系中的峰值漂移。*

- 脚本输出的关键数值（在温度窗口 1.5–3.5 内的峰值搜索结果）：

  - Local update:
    - $C_v$ 峰: T = 2.400, $C_v$ ≈ 1.03643
    - $\chi$ 峰: T = 1.500–1.900 (数据在低温处有波动；示例输出 $\chi$ ≈ 18.36 at T≈1.8)
    - 低温极限: E/N ≈ -2.0, |M|/N ≈ 1.0
    - 高温极限: E/N ≈ -0.2023, |M|/N ≈ 0.1662

  - Wolff:
    - $C_v$ 峰: T = 2.400, $C_v$ ≈ 1.02806
    - $\chi$ 峰: T ≈ 1.500 (示例输出 $\chi$ ≈ 23.38 at T≈1.5)
    - 低温极限: E/N ≈ -2.0, |M|/N ≈ 1.0
    - 高温极限: E/N ≈ -0.2013, |M|/N ≈ 0.1665


- 峰位与理论 $T_c\approx2.269$ 接近，但在有限尺寸 (6x6) 上存在显著的有限尺寸偏移與统计误差；本次数据的 $C_v$ 峰位出现在 T=2.4，略高于理论值，这应该属于常见的有限尺寸效应。
- Wolff 与 Local update 在比热曲线上的峰值高度与位置接近，但在磁化率 $\chi$ 上出现差异，且部分低温点出现较大波动（可能由样本量不足或测量/正规化问题导致）。
- Wolff 更新通常在临界附近能更有效地降低自相关时间，在相同采样开销下其误差通常更小；但本次数据差异也可能来自于每种算法在每个温度点的采样次数与热化设置不同，或测量间隔与自相关未充分处理。

## 1.4. 总结
- 本次 6x6 格点的测试中，Local update 与 Wolff 给出相似的 $C_v$ 峰位（约 T=2.4），但细节上存在差异，尤其是 $\chi$ 在低温处的波动。有限尺寸和统计误差是主要原因。
- 若要更可靠地估计临界行为，也许要：
  1. 增大格点尺寸（例如 16x16、32x32）并做有限尺寸标度分析；
  2. 对每个温度点增加独立样本与块引导（block bootstrap），并估计自相关时间以给出不确定度（error bars）；
  3. 对 Local update 增加测量间隔并记录自相关时间，或使用 Wolff 并以簇大小为度量评估有效采样；
- 如需更清晰可视化，在绘图时对同一温度点的两种方法使用微小水平抖动（jitter）或提示性误差条（error bars）。
